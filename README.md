# Estimation
Batch and Sequential estimation algorithms for small static or dynamic systems

**Batch**:
* MAP, MMSE
* MLE, LS
* Gauss-Newton
* Levenberg-Marquardt

**Sequential**:
* Kalman Filter
* Nonlinear Kalman Filter
* Extended Kalman Filter
* Unscented Kalman Filter
* Information Filter
* Square-Root Filter
* Square-Root Information Filter
* Particle Filter
* Multiple Model Filter
